Javascript is a single-threaded, synchronous, client side JIT compiled interpreted programming language. Briefly we can say, javascript runs a single line of code one at a time & It works from top to bottom serial wise, which we called it as synchronously. Javascript is also known for its client based operations. When a client opens a browser, he experiences dynamically compiled visuals. From html, css, javascript code a browser first works to parse the basic code from the coder. Completing the parse, the browser converts it to machine readable code. When this conversion is done, we see the web pages output in seconds. So, this is how javascript works.

Local Storage

  • Long time accessed or stored data
  • This storage data does not expire
  • Old data loads for previously created document
  • Data does continue after reloading the page
  • Stores string data

Session Storage

  • Short time accessed or stored data
  • This storage data expires session by session
  • New data created for every new sessioned document
  • Data not get continued after the page reloads
  • Stores string data
